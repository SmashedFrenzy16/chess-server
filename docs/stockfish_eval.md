# /stockfish_eval

Evaluates a chess position using the Stockfish engine.

| Name | Description | Required |
| ---- | ----------- | -------- |
| `position` | The position of the white and black piece(s) | Yes |

## Usage

### Example

```js
ws.send('/stockfish_eval e4 e5');
```

```text
1. e4 e5 0
```
